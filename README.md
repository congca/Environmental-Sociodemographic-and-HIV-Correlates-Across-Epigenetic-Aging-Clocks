# Environmental, Sociodemographic, and HIV Correlates Across Epigenetic Aging Clocks

Analysis code accompanying the manuscript:

Distinct Environmental and HIV-Related Determinants Across Multiple Epigenetic Aging Clocks. The study evaluates environmental exposures, sociodemographic characteristics, and HIV-related factors across multiple DNA methylation–based epigenetic aging clocks in the MACS/MWCCS cohort.

### Files

* `data clean.R` – data cleaning and preprocessing
* `generate table 1.R` – descriptive statistics and Table 1
* `Paper3_MultiClock_Epigenetic_Aging_stage1_clean.py` – primary analyses
* `DL_prediction_stage1_clean.py` – deep learning analyses
* `GENE_stage1_clean.py` – gene-related analyses

### Data

The individual-level data used in this study are not included in this repository. Access to MWCCS/MACS data requires approval through the MWCCS data access process. Environmental exposure data were obtained from publicly available sources described in the manuscript.

### Notes

Some file paths are specific to the original computing environment and should be updated before running the scripts.
